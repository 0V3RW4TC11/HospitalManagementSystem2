# Hospital Management System
## Overview
A full-stack web application built with ASP.NET Core MVC that follows a Clean (Onion) Architecture design. Data access and business logic are abstracted from application implementation.

Users of the application can manage and use three types of users:

* Admins: Manage all users, can create, disable and edit accounts.
* Doctors: Can create and view attendance forms for Patients.
* Patients: Can view attendances forms for their appointments.

## Getting started
1. Clone/Download the project from the 'Main' branch
2. Open the project with Visual Studio 2026 or later
3. Set Seeding/Menu as the startup project and run
4. Use the Seeder to create the database and seed all users
5. Use SQL Server object explorer to get a username from a user
6. Set Web as the startup project and run
